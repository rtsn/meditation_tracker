# meditation_tracker

```
usage: meditation_tracker.py [-h] [-f FILE] [-a ADD] [-s SET] [-c]

            A very simple meditation time tracker!
            --------------------------------
                This program loads an image
                with OpenCV and Python argparse!
            

optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  Path to tracker file.
  -a ADD, --add ADD     Add an interger number of minutes to the total
  -s SET, --set SET     Set total to n number of minutes
  -c, --compact         Compact output of the form days:hours:minutes.
```
